Helm Chart Repository

Continuous Monitoring

A Helm chart for deploying a full observability stack with Grafana, Prometheus, Loki, and exporters for node and Kubernetes state metrics. Built for clear defaults and fast cluster visibility.

View Helm index Chart details
Quickstart
helm repo add continuous-monitoring \
  https://firasmosbehi.github.io/continuous-monitoring-chart

helm install continuous-monitoring \
  continuous-monitoring/continuous-monitoring

Grafana default credentials: admin / admin

Stack

Metrics + Logs

Prometheus scrapes exporters, Grafana visualizes, Loki aggregates logs when enabled.

Dashboards

Exporter aware

Dashboards are provisioned only if their exporter is enabled, keeping Grafana clean.

Config

Values-driven

Toggle components with `values.yaml` and deploy only what your cluster needs.

Repo

Release ready

Packaged and indexed for Helm repo hosting on GitHub Pages.

Architecture

Grafana provides dashboards and datasources. Prometheus scrapes node-exporter and kube-state-metrics. Loki and its datasource are optional. Each component is independently toggled via values.
